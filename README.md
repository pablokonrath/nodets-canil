# nodets-canil

Repositorio criado para acompanhar o curso da b7web no modulo do curso node + typescripts

### Pré-requisitos globais

'npm i -g nodemon typescript ts-node'

## Instalação

'npm i '

## Para rodar o projeto

'npm run dev'
